## 🌟 Overview

**Techium** is a fully-featured HTML5 website built from scratch using semantic HTML elements, demonstrating best practices in web development. This project showcases a complete digital agency website with multiple pages, multimedia content, and a comprehensive style guide.

### 🎯 Project Goals
- Master semantic HTML5 structure
- Implement accessibility best practices
- Create responsive and interactive elements
- Build a comprehensive style guide
- Demonstrate multimedia integration

---

## ✨ Features

### 🏠 **Homepage Components**
- **Hero Section** - Eye-catching landing area
- **Services** - Interactive service listings
- **Portfolio** - Work showcase with images
- **About Us** - Company information with visuals
- **Latest News** - Blog-style content with author attribution
- **Testimonials** - Client feedback with avatar images
- **Contact** - Call-to-action section

### 📱 **Multi-page Navigation**
- `index.html` - Main homepage
- `about.html` - Detailed company information
- `latest_news.html` - News and blog content
- `contact.html` - Contact information and forms
- `article.html` - Individual article pages

### 🎨 **Style Guide**
Comprehensive HTML element demonstrations including:
- Typography (H1-H6, paragraphs, addresses)
- Lists (ordered, unordered, definition)
- Tables with proper accessibility
- Forms and interactive elements
- Multimedia (video, audio, iframes)
- Code blocks and formatting
- Blockquotes and citations

## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Text editor (VS Code recommended)
- Basic understanding of HTML5

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd html_advanced
   ```

2. **Open in browser**
   ```bash
   # Open the main homepage
   open index.html
   
   # Or view the style guide
   open styleguide.html
   ```

3. **Development**
   ```bash
   # Use a local server for best experience
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

## 🏆 Technical Highlights

### HTML5 Elements Used
```html
<!-- Semantic Structure -->
<header>, <main>, <section>, <article>, <aside>, <footer>

<!-- Content Elements -->
<h1>-<h6>, <p>, <blockquote>, <cite>, <address>

<!-- Lists & Tables -->
<ul>, <ol>, <dl>, <table>, <thead>, <tbody>

<!-- Multimedia -->
<video>, <audio>, <iframe>, <img>

<!-- Interactive -->
<details>, <summary>, <a>, <nav>

<!-- Forms & Input -->
<form>, <input>, <textarea>, <button>
```
